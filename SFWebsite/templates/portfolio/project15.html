Project 15: Scrum Methodology Using Kanban Chart

Introduction

This project shows the management of a software project using a Kanban Board.

The Kanban board is created as a Project in GitHub.

The screenshots show a template for user stories, a sample user story, the user stories for the project in the Product Backlog, estimates and labels and sprints assigned, and the user stories moving across the board to Done as the work is completed.



User Story Template

The user story template is created in GitHub by navigating to the Settings tab of your repository, then clicking on Set up Templates.

Once created it can be used to create user stories as issues in the Issues tab or directly on the Kanban board.



Kanban Board

The kanban board is created in GitHub by navigating to the Projects tab of your repository, then clicking on Set up Templates.

Once created it can be modified to have the columns you require.
