Project17: Implementing a Kubernetes Deployment

Introduction

This project demonstrates the implementation of a Kubernetes deployment.

The deployment is for a simple web application that displays a welcome message.

The application is deployed to a single node Kubernetes cluster provided by Docker Desktop.

Describe the Deployment

This screenshot shows the output of the kucectl describe command for the deployment.

Although not specified in the deployment yaml file the StrategyType is RollingUpdate which is the default update type.

In this strategy pods are replaced one at a time so the app is always available.
